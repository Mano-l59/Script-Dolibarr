# Procédure d'utilisation

## Les pré-requis

Pour exécuter les scripts il faut avoir une clé ssh fonctionnelle vers la machine dattier.

## Exécter les scripts

Pour faire une installation fonctionnelle vous pouvez exéctuer les scripts dans cet ordre : 1, 2, 3, 6, 4, 5.

Ou tout simplement exéctuer le script principal qui les exécute 1 à 1.

Pour configurer un nouvel utilisateur il vous faudra aussi suivre la Procédure : Configuration Dolibarr

## Comprendre le fonctionnement des scripts

Le comportement des scripts est expliqué via le nom de ces derniers mais aussi par les commentaires et les affichages présent à l'intérieur.
